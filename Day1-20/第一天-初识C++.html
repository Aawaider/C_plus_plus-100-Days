📘 一、什么是C++

C++ 是一种 通用型、面向对象的高级编程语言，由 Bjarne Stroustrup 于 1979 年在 C 语言的基础上发展而来。

它结合了 C语言的高性能 与 面向对象的抽象能力，是目前世界上使用最广泛的编程语言之一。

C++ 常被称为：

“既能贴近硬件，又能构建大型系统的语言。”

🚀 二、C++能做什么？

应用领域示例
系统开发操作系统、驱动程序、底层库
游戏开发Unreal Engine（虚幻引擎）、Unity底层
桌面应用Visual Studio、QT、Notepad++
嵌入式系统单片机、物联网设备、芯片固件
高性能计算科学计算、图像处理、仿真系统
金融与量化高频交易系统、风控引擎
人工智能底层TensorFlow、PyTorch 的核心都是C++实现

💡 一句话总结：

C++ 是一门“能干到底层、能写大型项目”的通用语言。

🧩 三、C++编译器（Compiler）是什么？

编译器是把我们写的 .cpp 源代码 翻译成计算机能执行的程序 的工具。

常见的C++编译器有：

编译器名称平台说明
GCC / G++Windows / Linux / macOS最常见、开源、免费
ClangmacOS / Linux编译速度快、输出信息友好
MSVCWindowsVisual Studio 自带的官方编译器

⚙️ 四、安装C++编译器

✅ Windows 用户

推荐安装：

  1. **MinGW（轻量）**
  2. 官网：[https://www.mingw-w64.org/](https://www.mingw-w64.org/)
  3. 安装后将 `bin` 路径加入环境变量。
  4. **Visual Studio（完整版）**
  5. 官网：[https://visualstudio.microsoft.com/](https://visualstudio.microsoft.com/)
  6. 安装时勾选「使用C++的桌面开发」。

✅ macOS 用户

  • 打开终端输入：
    • 
xcode-select --install

    自动安装 Xcode Command Line Tools（含 Clang 编译器）。

✅ Linux 用户

  • 打开终端输入：
    • 
sudo apt update
sudo apt install g++

💻 五、什么是IDE？

IDE（Integrated Development Environment，集成开发环境）

是一个 编写、调试、运行程序的完整工具平台。

相比普通文本编辑器，IDE 让编程更高效：

功能作用
语法高亮让代码更清晰
自动补全输入更快
一键编译运行不用手动输入命令
调试工具发现错误更容易

🧰 六、常用的C++ IDE推荐

IDE名称适用系统特点
**Visual Studio**Windows官方IDE，功能最强大
**CLion**Windows / macOS / Linux智能提示强大，界面现代
**Code::Blocks**Windows / Linux轻量，配置简单
**Dev-C++**Windows初学者经典工具
**VS Code + 插件**全平台轻便通用，需配置编译环境

💡 建议：

初学者推荐使用 **VS Code** 或 **Dev-C++**，上手快，界面简洁。
